# Team Routines (Example)

Use this file as a simple checklist for a bot's periodic tasks.
The bot will review the items and act if needed.

## Daily
- [ ] Check open tasks assigned to me and flag blockers.
- [ ] Scan recent notes for missing follow-ups.

## Weekly
- [ ] Review ongoing projects and update status summary.
- [ ] Verify key links/credentials are still valid (no secrets in this file).

## When prompted by user
- [ ] Perform the requested routine and report back clearly.
